# Reissue order token

Endpoint: POST /order/token/reissue

## Request fields (application/json):

  - `order_id` (string, required)
    order_id

## Response 200 fields (application/json):

  - `order_id` (string)
    Order id

  - `checkout_url` (string)
    Checkout url

  - `token` (string)
    Token
